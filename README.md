## 1. Реалізація (Implementation)

У цій роботі реалізовано програмну архітектуру для потокової обробки координатних даних, що імітує роботу мікроконтролера в реальному часі. Було імплементовано три типи цифрових фільтрів:

### Методи `update`:

```python
# SMA (Simple Moving Average) - Реалізація з оптимізацією O(1)
def update(self, x):
    # Якщо черга повна, віднімаємо старий елемент від загальної суми
    if len(self.q) == self.w:
        self.sum -= self.q[0]
    # Додаємо новий елемент у чергу та до суми
    self.q.append(x)
    self.sum += x
    # Повертаємо середнє значення
    return self.sum / len(self.q)

# EMA (Exponential Moving Average) - Рекурсивний фільтр
def update(self, x):
    # Якщо це перший відлік, ініціалізуємо значенням x
    if self.last is None:
        self.last = x
    else:
        # Рекурсивна формула: alpha * x + (1 - alpha) * last
        self.last = self.a * x + (1 - self.a) * self.last
    return self.last

# Median Filter - Нелінійний фільтр для ігнорування викидів
def update(self, x):
    # Додаємо x у чергу
    self.q.append(x)
    # Повертаємо медіану поточного вікна
    return np.median(self.q)
```

## Висновок

На основі проведених експериментів із придушення шумів у потоці даних у режимі реального часу (real-time) було визначено оптимальні типи фільтрів для різних сценаріїв використання:

* **(а) Для видалення збоїв сенсора (викидів):** Найкраще підходить **Медіанний фільтр (Median Filter)**. Як нелінійний фільтр, він володіє властивістю повністю ігнорувати поодинокі імпульсні викиди (outliers), на відміну від SMA та EMA, які "розмазують" такі аномалії по траєкторії.
* **(б) Для плавного ведення траєкторії:** Найкраще підходять лінійні фільтри **SMA (Simple Moving Average)** та **EMA (Exponential Moving Average)**. При цьому фільтр **EMA** є більш переважним для вбудованих систем та мікроконтролерів, оскільки він є рекурсивним і потребує мінімального обсягу пам'яті для зберігання стану.
* **Аналіз компромісів:** Дослідження підтвердило існування критичного балансу між якістю фільтрації та динамічними спотвореннями. Надмірне збільшення інерційності фільтрів призводить до ефекту "зрізання кутів" та значної фазової затримки (lag). У спектральному аналізі це проявляється зростанням помилки на низьких частотах (ліворуч), де амплітуда відхилення відфільтрованого сигналу від ідеалу стає більшою за рівень вхідного шуму через нездатність фільтра вчасно реагувати на маневри.
